OAMD - osu! Automatic Map Downloader

1

Introduction

Press Windows + R and run the code below on the machine you with to copy all of your maps names and IDs. If you installed osu! in a different directory, execute the 2nd code in your osu! directory

2

Code

Default directory:

cmd /k cd %localappdata%/osu!/Songs && dir /b | clip && exit

|

Custom directory:

cd Songs && dir /b | clip && exit

3

Info

If you want to add someones maps to your maps, paste their maps in the input field and your maps in the ignore field
